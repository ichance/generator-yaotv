# generator-yaotv [![NPM version][npm-image]][npm-url] 
> 摇电视项目素材包结构

## 安装

需预先安装[node.js](https://nodejs.org/)、[npm](https://www.npmjs.com/)

```bash
npm install -g yo
npm install -g generator-yaotv
```

然后执行项目目录生成命令:

```bash
yo yaotv
```
